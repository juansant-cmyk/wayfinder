# Design

UI/UX assets, wireframes, and design system docs for Wayfinder.

## Team ownership

| Area | Sprint 1 focus |
|------|----------------|
| Wireframes | Dashboard, auth, navigation (US-04) |
| Themes | Light / dark / accent presets (US-06) |
| Components | Shared patterns for frontend handoff |

## Layout

```
design/
├── wireframes/     # PNG/PDF exports, Figma links
├── themes/         # Color tokens, typography
├── assets/         # Icons, logos (export from Figma)
└── specs/          # Screen specs, spacing, accessibility notes
```

## Handoff to frontend

- Export assets to `design/assets/`
- Document theme tokens in `design/themes/`
- Link Figma files in this README when ready
